# 📚 Bibliotech

Uma moderna *landing page* para uma biblioteca digital e física, concebida para promover o acesso ao conhecimento, leitura e envolvimento da comunidade através de eventos e serviços. 

## 📖 Sobre o Projeto

O **Bibliotech** é um projeto *front-end* estático que simula a página principal de uma biblioteca contemporânea. O objetivo desta interface é proporcionar uma experiência de utilizador limpa e atrativa, facilitando a pesquisa por livros, a visualização de novas obras no catálogo, a inscrição em eventos e o conhecimento dos serviços oferecidos pelo espaço (como aluguer digital e salas de estudo).

## ✨ Funcionalidades

O website está dividido nas seguintes secções principais:
* **Cabeçalho (Header):** Menu de navegação rápido e botão de *login* para membros.
* **Hero Section:** Área de destaque com uma barra de pesquisa integrada para explorar o catálogo, autores ou códigos ISBN.
* **Novidades (New Arrivals):** Exposição dos livros adicionados recentemente ao acervo, com botões para requisição rápida.
* **Próximos Eventos (Upcoming Events):** Calendário interativo de atividades da biblioteca, como clubes de leitura e palestras com autores.
* **Serviços (How We Can Help):** Apresentação das facilidades modernas da biblioteca, incluindo recursos digitais, salas de estudo e apoio à investigação.
* **Rodapé (Footer):** Informações de contacto, horário de funcionamento e *links* rápidos.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído puramente com tecnologias web padrão, sem dependências de *frameworks* externas:
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização da página, utilizando *Flexbox* para o alinhamento de elementos e variáveis de cores consistentes (com foco em tons quentes e terrosos, como o `#BE580F`).
